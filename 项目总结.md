## ✅ 已实现功能

| 模块 | 功能 |
|------|------|
| **用户系统** 🆕 | 注册（用户名/密码/昵称）、登录、退出、Token 认证（7天有效） |
| **数据隔离** 🆕 | 每个用户只能查看和管理自己的任务 |
| 任务管理 | 创建、编辑、删除、查看任务 |
| 字段 | 标题、描述、课程、类型（作业/考试/实验/其他）、优先级（高/中/低）、截止时间 |
| 状态流转 | 未开始 → 进行中 → 已完成，超期自动标记"已逾期" |
| 智能提醒 | 24h 内到期黄色提醒 + 已逾期红色警告 |
| 筛选搜索 | 按状态、课程、类型筛选 + 关键字模糊搜索 |
| 日历视图 | 月视图展示每日任务分布，可切换月份 |
| 统计面板 | 全部/进行中/已逾期/已完成 数量统计 |
| 快捷键 | Ctrl+N 新建任务，Esc 关闭弹窗 |
| 响应式 | 手机、平板、电脑均可使用 |

## 📡 API 接口一览

| 方法 | 路径 | 说明 | 认证 |
|------|------|------|------|
| POST | `/api/auth/register` | 用户注册 | 无 |
| POST | `/api/auth/login` | 用户登录 | 无 |
| POST | `/api/auth/logout` | 退出登录 | Bearer Token |
| GET | `/api/auth/me` | 获取当前用户 | Bearer Token |
| GET | `/api/tasks` | 任务列表 | Bearer Token |
| POST | `/api/tasks` | 创建任务 | Bearer Token |
| GET | `/api/tasks/{id}` | 获取任务 | Bearer Token |
| PUT | `/api/tasks/{id}` | 更新任务 | Bearer Token |
| DELETE | `/api/tasks/{id}` | 删除任务 | Bearer Token |
| GET | `/api/tasks/calendar` | 日历范围查询 | Bearer Token |
| GET | `/api/reminders` | 获取提醒 | Bearer Token |
| GET | `/api/stats` | 统计数据 | Bearer Token |
