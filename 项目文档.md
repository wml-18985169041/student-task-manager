## 1. 项目概述

### 背景

学生日常学习任务（课程作业、实验报告、考试安排等）分散在笔记本、便签、聊天记录中，容易遗漏。截止时间临近时缺乏有效提醒，逾期后才发现未提交。

### 目标

构建一个支持多用户的轻量级 Web 应用，提供注册登录、任务创建、优先级设置、截止时间提醒、分类整理、日历视图等功能，将零散的学习任务集中管理，提高学习安排的系统性和执行效率。

### 适用场景

- 大学生管理多门课程的作业与实验
- 跟踪考试复习进度
- 规划日常学习日程
- 避免遗漏截止日期

## 3. 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| **后端框架** | FastAPI | 0.115.6 |
| **密码加密** | passlib (bcrypt) | 1.7.4 |
| **数据校验** | Pydantic | 2.10.3 |
| **服务器** | Uvicorn | 0.34.0 |
| **数据库** | SQLite | 内置 |
| **前端** | HTML5 + CSS3 + Vanilla JS | — |
| **Python** | 3.10+ | — |

### 选型理由

- **FastAPI**：异步高性能，自动生成 OpenAPI 文档，类型安全，依赖注入方便实现认证
- **passlib + bcrypt**：安全可靠的密码哈希方案
- **SQLite**：零配置、免安装、单文件存储，适合轻量级应用
- **Token 认证**：无状态，前端 localStorage 存储，实现简单
- **原生前端**：无框架依赖，加载快，部署简单，适合单页应用场景

## 5. 快速开始

### 前置要求

- Python 3.10+
- pip

### 步骤 1：进入项目目录

```bash
cd student-task-manager
```

### 步骤 2：安装依赖

```bash
pip install -r requirements.txt
```

### 步骤 3：启动服务

```bash
python main.py
```

或直接使用 uvicorn：

```bash
uvicorn main:app --host 0.0.0.0 --port 8080
```

### 步骤 4：打开浏览器

访问 **http://127.0.0.1:8080** 即可使用。

首次使用需注册账号。

## 6. 环境变量

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `TASK_APP_HOST` | `0.0.0.0` | 服务监听地址 |
| `PORT` 或 `TASK_APP_PORT` | `8080` | 服务端口（`PORT` 优先级更高，兼容 Render） |
| `TASK_DB_PATH` | `student_tasks.db` | SQLite 数据库文件路径 |
| `TASK_REMIND_HOURS` | `24` | 提前多少小时提醒即将到期任务 |
| `TASK_FRONTEND_ORIGIN` | `*` | 允许跨域的前端地址 |

## 7. API 接口文档

### 基础信息

- **Base URL**: `http://127.0.0.1:8080/api`
- **Content-Type**: `application/json`
- **OpenAPI 文档**: 启动服务后访问 `http://127.0.0.1:8080/docs`

---

### 7.1 认证接口

#### 注册

```
POST /api/auth/register
```

**请求体：**

```json
{
  "username": "zhangsan",
  "password": "123456",
  "nickname": "张三"
}
```

**响应（201）：**

```json
{ "id": 1, "message": "注册成功，请登录" }
```

#### 登录

```
POST /api/auth/login
```

**请求体：**

```json
{
  "username": "zhangsan",
  "password": "123456"
}
```

**响应：**

```json
{
  "token": "abc123...",
  "user": { "id": 1, "username": "zhangsan", "nickname": "张三" },
  "message": "登录成功"
}
```

#### 获取当前用户

```
GET /api/auth/me
Authorization: Bearer <token>
```

#### 退出登录

```
POST /api/auth/logout
Authorization: Bearer <token>
```

---

### 7.2 任务接口

> 以下所有接口需携带 `Authorization: Bearer <token>` 请求头。

#### 获取任务列表

```
GET /api/tasks
```

**查询参数：**

| 参数 | 类型 | 说明 |
|------|------|------|
| `status` | string | 按状态筛选 |
| `course` | string | 按课程名称筛选 |
| `task_type` | string | 按类型筛选 |
| `priority` | string | 按优先级筛选 |
| `search` | string | 模糊搜索标题和描述 |

**响应示例：**

```json
{
  "count": 2,
  "tasks": [
    {
      "id": 1,
      "user_id": 1,
      "title": "完成高数作业第五章",
      "description": "1-10题，注意证明题步骤",
      "course_name": "高等数学",
      "task_type": "作业",
      "priority": "高",
      "deadline": "2026-06-18T15:00",
      "status": "进行中",
      "created_at": "2026-06-16 18:11:47",
      "updated_at": "2026-06-16 18:11:47"
    }
  ]
}
```

#### 获取单个任务

```
GET /api/tasks/{task_id}
Authorization: Bearer <token>
```

#### 创建任务

```
POST /api/tasks
Authorization: Bearer <token>
```

**请求体：**

```json
{
  "title": "完成高数作业",
  "description": "第五章 1-10题",
  "course_name": "高等数学",
  "task_type": "作业",
  "priority": "高",
  "deadline": "2026-06-18T15:00",
  "status": "未开始"
}
```

**响应（201）：**

```json
{ "id": 2, "message": "任务创建成功" }
```

#### 更新任务

```
PUT /api/tasks/{task_id}
Authorization: Bearer <token>
```

#### 删除任务

```
DELETE /api/tasks/{task_id}
Authorization: Bearer <token>
```

#### 日历范围查询

```
GET /api/tasks/calendar?start_date=2026-06-01&end_date=2026-06-30
Authorization: Bearer <token>
```

#### 获取提醒

```
GET /api/reminders
Authorization: Bearer <token>
```

#### 获取统计

```
GET /api/stats
Authorization: Bearer <token>
```

### 错误响应

| HTTP 状态码 | 含义 |
|-------------|------|
| `200` | 成功 |
| `201` | 创建成功 |
| `400` | 请求参数错误 |
| `401` | 未登录或 token 过期 |
| `404` | 任务不存在 |
| `422` | 请求体校验失败 |

## 9. 数据库设计

### 表：`users`

```sql
CREATE TABLE users (
    id            INTEGER PRIMARY KEY AUTOINCREMENT,
    username      TEXT    NOT NULL UNIQUE,
    password_hash TEXT    NOT NULL,
    nickname      TEXT    DEFAULT '',
    created_at    TEXT    DEFAULT (datetime('now','localtime'))
);
```

### 表：`sessions`

```sql
CREATE TABLE sessions (
    token      TEXT PRIMARY KEY,
    user_id    INTEGER NOT NULL REFERENCES users(id),
    created_at TEXT DEFAULT (datetime('now','localtime')),
    expires_at TEXT NOT NULL
);
```

### 表：`tasks`

```sql
CREATE TABLE tasks (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id     INTEGER NOT NULL REFERENCES users(id),
    title       TEXT    NOT NULL,
    description TEXT    DEFAULT '',
    course_name TEXT    DEFAULT '',
    task_type   TEXT    DEFAULT '其他',
    priority    TEXT    DEFAULT '中',
    deadline    TEXT    NOT NULL,
    status      TEXT    DEFAULT '未开始',
    created_at  TEXT    DEFAULT (datetime('now','localtime')),
    updated_at  TEXT    DEFAULT (datetime('now','localtime'))
);
```
